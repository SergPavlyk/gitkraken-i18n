## GitKraken Translations

This repository provides GitKraken translation and localization files for multiple languages.

Available language packs:

- GitKraken German translation (`de-DE`)
- GitKraken English localization (`en-US`)
- GitKraken Spanish translation (`es-ES`)
- GitKraken Japanese translation (`ja-JP`)
- GitKraken Portuguese (Brazil) translation (`pt-BR`)
- GitKraken Russian translation (`ru-RU`)
- GitKraken Ukrainian translation (`uk-UA`)
- GitKraken Chinese translation (`zh-CN`)

These GitKraken language files can be used to replace the default `strings.json` and localize the GitKraken interface.

## Встановлення перекладу

Файл, що відповідає за мову інтерфейсу GitKraken:

### Windows

```text
%LOCALAPPDATA%\gitkraken\app-X.X.X\resources\app.asar.unpacked\src\strings.json
```

### macOS

```text
/Applications/GitKraken.app/Contents/Resources/app.asar.unpacked/src/strings.json
```

### Встановлення

1. Відкрийте каталог `locales` у репозиторії.
2. Виберіть потрібну локаль.
3. Скопіюйте файл `strings.json`.
4. Замініть оригінальний файл у каталозі GitKraken.
5. Перезапустіть GitKraken.
